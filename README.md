# Руководство по сборке и использованию чекера лабораторных работ (Модуль Docker)

Данное руководство описывает шаги по компиляции приложения `Z69`, настройке окружения и тестированию 10-й лабораторной работы.

---

## Часть 1. Компиляция приложения

Чтобы превратить исходный код в один исполняемый файл `.exe`, внимательно выполните следующие шаги:

1. **Установите зависимости**
   Чтобы программа могла работать, ей нужны дополнительные библиотеки.
   - Зайдите в папку с проектом `Z69`.
   - Найдите файл с названием `install_deps.bat`.
   - Кликните по нему **правой кнопкой мыши** и выберите пункт **Запуск от имени администратора** (или просто дважды кликните).
   - В открывшемся черном окне дождитесь надписи `Установка завершена` и нажмите любую клавишу для закрытия окна.

2. **Скомпилируйте приложение**
   - Откройте терминал или командную строку (введите `cmd` в адресной строке проводника, находясь в папке `Z69`, и нажмите Enter).
   - Напечатайте команду:
     `python build.py`
     и нажмите клавишу **Enter**.
   - Дождитесь завершения сборки (1-2 минуты). В папке проекта появится новая папка `dist`. Внутри неё лежит готовый файл **`Z69.exe`**.

---

## Часть 2. Установка Docker и настройка WSL

Для работы проверок на компьютере должен быть запущен Docker Desktop.

### Шаг 1. Проверка версии WSL
Откройте **Windows PowerShell** и введите команду:
`wsl -l -v`
- Если в ответ выводится таблица, где в колонке VERSION стоит **2**, значит у вас уже всё настроено! **Пропустите шаги 2-5 и сразу переходите к "Шагу 6. Установка Docker Desktop".**
- Если команда выдает ошибку, пишет, что `wsl` не распознано, или версия стоит **1**, переходите к Шагу 2 ниже.

### Шаг 2. Включение компонентов Windows вручную
- Нажмите **правой кнопкой мыши** по меню **"Пуск"** и выберите **Windows PowerShell (администратор)**.
- Скопируйте, вставьте (правой кнопкой мыши) и выполните (нажав Enter) первую команду:
  `dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart`
- Дождитесь 100%. Затем скопируйте и выполните вторую команду:
  `dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart`
- Дождитесь 100%. Закройте синее окно.

### Шаг 3. Обязательная перезагрузка
Обязательно **перезагрузите компьютер**, иначе ничего не сработает.

### Шаг 4. Обновление ядра Linux (Ручной метод)
- Откройте браузер, скопируйте эту ссылку, вставьте в адресную строку и нажмите Enter, чтобы скачать файл обновления:
  `https://wslstorestorage.blob.core.windows.net/wslblob/wsl_update_x64.msi`
  *(Если ссылка не скачивается, вбейте в поиск Яндекса или Гугла: "Скачать пакет обновления ядра Linux WSL2" и скачайте установщик с официального сайта Microsoft).*
- Запустите скачанный файл `wsl_update_x64.msi` и нажимайте "Next", пока установка не закончится.

### Шаг 5. Переключение на WSL 2
- Снова откройте **Windows PowerShell (администратор)**.
- Вставьте команду и нажмите **Enter**:
  `wsl --set-default-version 2`
- Закройте окно.

### Шаг 6. Установка Docker Desktop
- Скачайте установщик Docker. Скопируйте эту ссылку и вставьте в адресную строку браузера:
  `https://desktop.docker.com/win/main/amd64/Docker%20Desktop%20Installer.exe`
  *(Или вбейте в поиск: "Docker Desktop download for Windows" и скачайте с официального сайта).*
- Запустите скачанный файл. В первом окне обязательно убедитесь, что стоит галочка **Use WSL 2 instead of Hyper-V**. Нажмите "OK".
- После установки запустите Docker Desktop с ярлыка на рабочем столе. В левом нижнем углу программы должен появиться зеленый значок кита (надпись `Engine running`).

## Часть 3. Проверка 10-й лабораторной работы

Файлы для 10-й лабораторной работы уже лежат в папке `10` внутри вашего проекта `Z69`.

1. **Запустите чекер**
   Зайдите в папку `Z69\dist` и запустите скомпилированный `Z69.exe`.

2. **Откройте консоль Docker**
   В левой панели приложения нажмите на **вторую иконку сверху** (синий кит).

3. **Укажите номер лабораторной работы**
   Кликните в черное поле ввода внизу консоли, введите команду `lab lab10` и нажмите **Enter**.

4. **Укажите путь к файлам**
   - В левом меню иконок нажмите на **третью иконку сверху** (желтая папка).
   - В открывшемся окне найдите папку с проектом `Z69`, откройте её, выделите папку `10` и нажмите "Выбор папки".

5. **Запустите проверку**
   В поле ввода консоли напечатайте команду `check` и нажмите **Enter**. Программа соберет среду и выполнит проверки. Зеленый цвет `[PASSED]` означает успешное прохождение пункта.
